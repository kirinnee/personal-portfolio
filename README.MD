# Personal Portfolio

My personal portfolio that I don't feel like managing nor updating.


# Getting Started

## Development
To run a local dev server
```bash
$ npm run dev
```
or 
```bash
$ yarn dev
```
## Unit Test
Run unit test
```bash
$ npm run test
```
or
```bash
$ yarn test
```
